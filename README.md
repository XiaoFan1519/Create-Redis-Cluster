# Create-Redis-Cluster
Creating redis cluster based on docker

# 版本要求
Docker Version [18.06.1-ce)

Redis(redis-cli) [5.0.0)

# 说明
默认以 REDIS REDIS_1 REDIS_2 REDIS_3 REDIS_4 REDIS_5 命名将要创建的6个容器

暂时只能在docker网络内部使用该集群
